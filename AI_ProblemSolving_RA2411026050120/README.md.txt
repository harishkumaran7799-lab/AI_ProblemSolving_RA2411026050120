# Smart Navigation System (BFS & DFS)

## 📌 Problem Description
This project simulates a navigation system that finds paths between locations using graph algorithms.

## ⚙ Algorithms Used
- BFS (Breadth First Search) → Finds shortest path
- DFS (Depth First Search) → Finds any path

## ▶ How to Run
1. Open terminal
2. Run:
   python main.py

## 📊 Sample Output
Start: A  
Goal: C  

BFS Path: A → B → C  
DFS Path: A → B → C  

## 📈 Comparison
| Feature | BFS | DFS |
|--------|-----|-----|
| Path | Shortest | Not guaranteed |
| Nodes Explored | More | Less |
